Please follow and complete the installation and configuration instructions for the plugin [here](https://curatora.io/how-to-connect-and-publish-to-wordpress-site-from-curatora-app/).

== Installation ==
  
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
